# Web 安全测试指南检查清单

此文件夹中包含一个 Excel 文件，提供以下工作表：

- _测试检查清单_ - 根据指南中概述的每个"测试"提供简单的进度跟踪。
- _总结发现_ - 便于创建测试结果和潜在建议的表格。
- _风险评估计算器_ - 一个下拉驱动的表单，用于计算可能性和影响评分，以及定性的整体风险评级。
- _参考资料_ - 提供计算器所基于的列表/集合。

**注意：** 当前的（Excel）检查清单基于 OWASP 测试指南 v4.2，因为 v5 内容仍在开发中。

## Google 表格模板

以下说明可用于将检查清单电子表格模板直接复制到新的 Google 表格中，而无需先在本地保存文档。

1. 访问此 [Google 电子表格模板](https://docs.google.com/spreadsheets/d/1csiYqA3DXhpz69K2JCLKN4H-kzkRFlFi/copy?copyCollaborators=false&copyComments=false&title=WSTG+Checklist)
2. 点击 `制作副本` 按钮。这将在您登录的 Google 云端硬盘中创建一个新的检查清单。
3. 您现在应该在 Google 表格中拥有一个完全填充且功能齐全的 Web 安全测试指南检查清单，包含上述四个标签页。
